## Project Description
This repository contains the implementation of a Jenkins-based Continuous Integration (CI) setup as part of a DevOps lab assignment.

## Contents
- Jenkins Freestyle Job
- Jenkins Multibranch Pipeline
- Jenkinsfile
- CI/CD concepts and architecture
- Installation and configuration documentation

## Jenkins Jobs Implemented
### Job 1: Freestyle CI Job
- Source Code Management: GitHub
- Build Triggers: Poll SCM and Webhook
- Build Steps: Windows batch commands and Python script
- Post-build Actions: Artifacts archiving and build results

### Job 2: Multibranch CI Pipeline
- Automatic branch discovery
- Jenkinsfile-based pipeline
- Separate handling for main branch
